CSS NOTES - 2/5/18

What is CSS

  • Cascading style sheets
  • defines a webpages appearance
  • the css will be in the head of the html file or in its own file
  • when u make CSS its own file, then it can be used on many many other files
  • HTML MUST BE PROPERLY FORMATTED FOR THIS TO WORK


History

  • there were9 different style sheets and things kept needing to be changed
  • it is up to the browser to determine if it is going to support the css (each browser may or may not support something)


Selectors

  • specific elements: h1, h2,- h6, body, p,
  • id -- identifier tag unique to the document - generally only want to use them once
  • class - indentifier that groups multiple elements ina document, and can be used multiple times
